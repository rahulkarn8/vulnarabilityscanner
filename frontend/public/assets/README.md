# Assets Directory

Place your logo and other static assets here.

## Logo File

Please place your DAIFEND AI CYBERSECURITY logo file here with one of these names:
- `daifend-logo.png`
- `daifend-logo.svg`
- `daifend-logo.jpg`

The logo should be:
- Recommended size: 200-300px width (height will auto-scale)
- Format: PNG, SVG, or JPG
- Transparent background (PNG) or white background

If using a different filename, update the `src` attribute in `frontend/src/components/Header.tsx`.
